AstroPath Dev UI

Developer-facing pages that exercise every backend endpoint end-to-end. Language picker in the top bar switches the API locale across the whole app. Production user flow lives in the Flutter mobile_app; these pages are for testing the backend.

How the language picker works

Each page calls the backend with ?lang=<code> and an Accept-Language header. The resolved locale is echoed back in meta.locale of every response. Try switching the picker in the top bar and reloading any page.
